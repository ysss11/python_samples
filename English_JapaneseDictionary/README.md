## 英和辞書から XXX が含まれている語句を抽出するプログラム
### 内容
```md
英和辞書データからwordに指定された語句が含まれている単語を抽出するプログラム
wordに検索する単語を入力する
```
### 使い方
```md
python word_search.py
```
